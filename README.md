# A Button That Can Do Anything  
### As Long As You're Asleep

Make a habit of opening the app, comparing the numbers to check if you're dreaming, and pressing the button while trying to do something you want to happen. 
Once the habit is built, occasionally, you will do this while you're dreaming and the number will not match, when this happens the button really can do anything you want it to.

May take practice. Will take time. No guarantees. Have fun.
